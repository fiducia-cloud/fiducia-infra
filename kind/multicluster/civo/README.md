# Civo emulator overlay

This Kustomize overlay represents the Civo member of the local three-cluster
Raft topology. `topology.env` is generated by `../up.sh` with the current Kind
node addresses and peer URLs; `kustomization.yaml` combines it with the shared
emulator resources in `../common`.

Apply the complete environment through `../up.sh`, not this directory alone, so
all three peer lists are generated from the same live Docker network state.
